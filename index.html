OVRTools

A collection of open-source tools and libraries for OpenVR focused on making VR UX better.

About

There are many luxuries we take for granted in desktop-land but often forget about in VR. We're trying to fill that gap by creating a collection of open-source tools and libraries for OpenVR that make your life in VR easier.

At the moment, we are focused on creating useful VR overlays that can be used in any VR application. We're also making libraries to support the creation of these overlays and make working with the OpenVR API easier by handling all the lower-level details for you, and exposing an idiomatic high-level API. We hope this empowers everyone to easily create and publish VR overlays.

Contribute

If you would like to help us make the VR user experience better, we accept different types of contributions to our projects, including but not limited to:

  • Bug fixes
  • New features
  • Documentation
  • Testing
  • Feature requests

Additionally, if you have an existing project that you think would feel at home in our collection (and get all the support that comes with it) please reach out to us! We'd love to adopt it.
